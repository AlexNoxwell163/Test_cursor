# Калькулятор стоимости услуг (V2)

Учебный проект: **index.html** + **style.css** + **script.js**, без npm.

**Услуги:** клининг, ремонт квартиры, уход за людьми. К любой услуге автоматически добавляется выезд (500 ₽).

## Запуск

Откройте в браузере файл **[index.html](./index.html)** (двойной клик или перетащите в окно Chrome / Firefox / Edge).

Сервер и Node.js не нужны.

## Где править цены

В `script.js` — объект `PRICES` и логика расчёта.
