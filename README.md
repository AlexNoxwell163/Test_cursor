# Lumière — сайт записи в парикмахерскую

Современный одностраничный сайт парикмахерской на **Next.js 15 (App Router)** + **TypeScript** + **Tailwind CSS v4**. Включает онлайн-запись с валидацией формы и сохранением в `localStorage`, страницы услуг, мастеров, галереи, отзывов и контактов.

## Технологии

- [Next.js 15](https://nextjs.org/) (App Router, React 19)
- TypeScript
- Tailwind CSS v4 (через `@tailwindcss/postcss`)
- [react-hook-form](https://react-hook-form.com/) + [zod](https://zod.dev/) — валидация формы
- [lucide-react](https://lucide.dev/) — иконки
- Шрифты Inter и Playfair Display через `next/font`

## Структура страниц

| Маршрут | Описание |
| --- | --- |
| `/` | Главная: hero, преимущества, превью услуг, отзывы, CTA |
| `/services` | Все услуги с фильтром по категории |
| `/masters` | Карточки мастеров |
| `/booking` | Форма онлайн-записи |
| `/booking/success?id=...` | Подтверждение записи |
| `/gallery` | Галерея работ |
| `/reviews` | Отзывы клиентов |
| `/contacts` | Адрес, телефон, часы работы, карта |

```bash

```

## Как работает запись

1. Пользователь выбирает услугу, мастера (фильтруется по специализации), дату (от сегодня до +30 дней) и слот времени (10:00–20:00 с шагом 30 минут).
2. Заполняет имя, телефон и опциональный комментарий.
3. Форма валидируется через **zod**: имя ≥ 2 символов, телефон по regex, обязательные поля.
4. При сабмите запись сохраняется в `localStorage` (ключ `barbershop:bookings`) и пользователь попадает на `/booking/success?id=...` с детализацией.

Бэкенда нет — данные услуг, мастеров, галереи и отзывов мокаются в `src/data/`. Для подключения реального API замените функции в `src/lib/storage.ts` и mock-файлы на запросы к серверу.

## Структура проекта

```
src/
  app/                   # маршруты (App Router)
  components/            # переиспользуемые UI-компоненты
  data/                  # mock-данные: услуги, мастера, отзывы, галерея
  lib/                   # утилиты: zod-схема, localStorage, тайм-слоты
  types/                 # общие TypeScript-типы
```

## Кастомизация

- Цветовая палитра задаётся CSS-переменными в `src/app/globals.css` (фон, акцент, текст).
- Услуги, мастера и отзывы редактируются в `src/data/*.ts`.
- Часы работы и шаг слотов — в `src/lib/timeSlots.ts` (`WORK_START_HOUR`, `WORK_END_HOUR`, `SLOT_STEP_MINUTES`).
