# Калькулятор стоимости услуг

Учебный проект: клиент выбирает параметры — приложение показывает разбивку цены.

**Услуги:** клининг, ремонт, фриланс.

## Запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Проверки

```bash
npm run lint
npm run typecheck
npm run build
```

## Документация для обучения

См. **[PROJECT.md](./PROJECT.md)** — карта папок, поток данных, чеклист.

## Стек

Next.js 15 (App Router), TypeScript, Tailwind CSS v4.
